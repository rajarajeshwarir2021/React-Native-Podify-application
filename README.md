# 🎧 Podify – Podcast Explorer App

Podify is a modern React Native application that allows users to discover, explore, and listen to podcasts. It provides a clean and intuitive interface for browsing trending podcasts, searching by category, and previewing episodes.

---

## 🚀 Features

* 🔍 Search podcasts by name or category
* 📈 Browse trending and popular podcasts
* 🎧 Listen to episode previews
* ⭐ Save podcasts to favorites
* 📂 Explore podcasts by categories (Tech, Business, Health, etc.)
* 💾 Persistent data storage (favorites)

---

## 🛠️ Tech Stack

* **Frontend:** React Native (Expo)
* **API:** Listen Notes API / Spotify API
* **State Management:** Context API / Redux (optional)
* **Storage:** AsyncStorage
* **Audio Playback:** Expo AV

## ⚙️ Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/rajarajeshwarir2021/React-Native-Podify-application.git
   cd React-Native-Podify-application
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the app:

   ```bash
   npx expo start
   ```

4. Run on device/emulator:

    * Press `i` for iOS simulator
    * Press `a` for Android emulator
    * Scan QR code with Expo Go

---

## 🔑 Environment Variables

Create a `.env` file in the root directory and add your API key:

```env
API_KEY=your_api_key_here
```
